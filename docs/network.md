# How to run Server/Client code

1. create 2 terminals
2. cd to test directory on both
3. compile with "g++ server.cpp -o s.exe" in 1st terminal
4. compile with "g++ client.cpp -o c.exe" in 2nd terminal
5. run "./s.exe" in 1st terminal
6. run "./c.exe" in 2nd terminal
7. send messages to server
8. to exit, press enter
